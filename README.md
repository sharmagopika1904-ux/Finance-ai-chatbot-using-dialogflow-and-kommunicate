# 🤖 Finance AI Chatbot Using Dialogflow & Kommunicate

An AI-powered Finance Assistant Chatbot developed using **Google Dialogflow ES** and **Kommunicate**. The chatbot helps users understand financial concepts, calculate ratios, and receive guidance on investment planning, loan decisions, budgeting, and tax-saving strategies through a conversational interface.

## 📌 Project Overview

The Finance AI Chatbot is a no-code/low-code conversational AI solution designed for finance students, professionals, and beginners seeking quick financial guidance.

The chatbot leverages:

- **Dialogflow ES** for Natural Language Processing (NLP) and intent recognition
- **Kommunicate** for chatbot deployment and user interaction
- Financial knowledge base for educational and advisory responses

The system recognizes user queries and provides relevant financial explanations, formulas, examples, and recommendations. Dialogflow uses intents and training phrases to map user queries to predefined responses. Kommunicate enables seamless chatbot deployment and human-agent handoff capabilities. :contentReference[oaicite:0]{index=0}
